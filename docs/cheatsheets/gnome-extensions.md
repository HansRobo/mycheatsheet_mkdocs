### Extension Sync
GitHubのgist経由でGNOME拡張機能の設定ファイルを同期できるプラグイン
拡張機能の同期はありそうでなかった機能だったのでとても便利に使わせてもらっている．

同期するには設定で以下の2点が必要
- GitHub GistのID
	- 初めての場合[ここ](https://gist.github.com/)から作る
	- 入力するのは`https://gist.github.com/username/XXXXX`の`XXXX`
- GitHubのパーソナルアクセストークン
